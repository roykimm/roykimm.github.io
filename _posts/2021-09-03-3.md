### 오라클 dblink 사용 ###

```
CREATE (PUBLIC) DATABASE LINK (데이터 베이스 링크명)
CONNECT TO (원격DB 유저명)
INDENTIFIED BY (원격DB 패스워드)
USING (TNS등록된 해당 ALIAS)
```

```
SELECT * FROM OPP.USER@DBMS_ORIGIN

CREATE TABLE USER AS SELECT * FROM USER@DBMS_ORIGIN
```
